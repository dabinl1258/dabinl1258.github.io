# UART Protocol Spec
* Serial Protocol
* Asynchronous Full Duplex
* Send LSB Bit First
* IDLE      : High Signal
* Start Bit : Low  Signal 1 Tick
* Stop  Bit : High Signal 1.5 ~ 2 Tick
