## 2.자신만 1인 비트를 표현하는 매크로
### 00000001 , 00000100 등
```c
#define BIT(i) (0x01 << i)
```


### 예를 들어서 0x01 인경우(0번째 비트가 1)
```c
BIT(0); // 0x01;
```
## 3.자신만 0비트인 경우를 표현하는 매크로
### 11110111 , 11011111 등
```c
#define NOTBIT(i) ~BIT(i)
```
### 11110111 을 표현
```c
NOTBIT(3); // 0xF7;
```
